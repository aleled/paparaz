Features

Everything you need.
Nothing you don't.

Built for developers, designers, and support teams who live in screenshots.

🎯
Pixel-Perfect Capture

Magnifier loupe with pixel grid, crosshair and hex color readout. Arrow-key precision nudging. Captures at full physical resolution on HiDPI/Retina displays — no blur, ever.

HiDPILoupeMulti-monitor
✏️
Rich Annotation Toolkit

20+ tools: arrows, curved arrows, rectangles, ellipses, freehand pen, brush, highlight, text, blur/masquerade, fill, eraser, eyedropper, crop, and slice.

20+ toolsLayersUndo/Redo
🔢
Smart Number Markers

Auto-incrementing callout markers in 4 styles — numeric (1 2 3), alpha (a b c), roman (I II III), and boxed. Reset counter any time. Font auto-scales for long labels.

4 stylesAuto-incrementReset
💾
Project File Format

Save your entire annotated session as a .papraz file — background image + all elements preserved. Reopen and keep editing any time.

.papraz filesCtrl+Shift+P
🔍
Built-in OCR

Extract text from any region of your screenshot using Windows built-in OCR (WinRT). Results are instantly copied to clipboard with one click.

WinRT OCRClipboard
⚡
Always-on Tray App

Lives quietly in your system tray. Global hotkeys for region (PrtScn), window (Alt+PrtScn), fullscreen, and repeat-last. Auto-saves every capture.

Global hotkeysAuto-saveRecent captures
